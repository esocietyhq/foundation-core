# Input Schema — Foundation

Foundation begins by capturing raw ideas in a structured way.

---

## Required Inputs

### 1. Idea Name
What is the name of the idea, product, business, or system?

### 2. Purpose
What is it trying to achieve?

### 3. Audience
Who is it for?

### 4. Problem
What problem does it solve?

### 5. Value
What useful outcome does it create?

### 6. Mechanism
How does it work?

### 7. Current State
Does it exist yet, or is it only an idea?

### 8. Constraints
What limits, risks, missing pieces, or unknowns exist?


## Input Goal

The goal is not to make the idea sound impressive.

The goal is to expose what it actually is.
